## Project Description

An application allows to fetch useful data about recent meetings activity. It stores data at Mongo's cluster

Project stack:
- Python 3.8
- Selenium - Browser navigating
- BeautifulSoup - parsing html
- MongoDB - Database
<br/>

##### To run app in development, install required packages via pipenv and type:
```
python3 scrap.py
```
<br/>

You also need working browser driver. Find out more:  
`https://selenium-python.readthedocs.io/installation.html#drivers`
